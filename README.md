# 🌐 Applications Development

This repository contains a comprehensive collection of **web development and programming projects**, systematically organized from **basic HTML concepts to interactive JavaScript applications and Python fundamentals**.

It is designed to serve as a **complete learning path for Application Development**, covering:

- HTML basics and mini projects
- Layout design and forms
- Frameset-based web structures
- Multi-step forms and dashboards
- JavaScript interactions
- Python programming fundamentals

Each folder represents a **progressive level of learning**, making it ideal for:

- 📘 Academic lab work
- 💻 Practice and revision
- 🎯 Interview and project preparation

## 🚀 How to Use

▶️ **Run HTML/CSS/JS Projects**

1. Open the project folder
2. Open `index.html` in your browser:

```text
    open index.html
```

▶️ **Run Python Notebooks**

```bash
    jupyter notebook
```

Open `.ipynb` files and run them step-by-step.
