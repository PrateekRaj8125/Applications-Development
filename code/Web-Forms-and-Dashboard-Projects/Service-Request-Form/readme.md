# Service Request & Support Ticket System

This folder contains an HTML project that implements an **online service request and support ticket form** with modern HTML5 elements and CSS styling.

The application allows users to submit detailed issue reports with structured input fields and validation.

---

## 📂 Program Structure

| S. No. | Description | File |
| --- | --- | --- |
| 1. | Main service request form | [`index.html`]() |
| 2. | Confirmation/response page | [`submit.html`]() |
| 3. | Styling for form and UI | [`style.css`]() |

---

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ```

2. Fill out the form and submit
3. View confirmation page

---

## 🧩 Description

The application includes:

- 📝 **User Information Section**
  - Name, email, contact, ticket ID

- ⚙️ **Issue Details**
  - Subject, date, problem type, categories
  - Description with progress indicators

- 🧰 **Service Information**
  - Multi-select services and file upload

- 🚨 **Priority & Consent**
  - Priority selection (radio buttons)
  - Urgency slider and confirmation checkbox

- 🎨 **UI Styling**
  - Clean and responsive form design using CSS

- ✅ **Submission Flow**
  - Redirects to confirmation page after submission
