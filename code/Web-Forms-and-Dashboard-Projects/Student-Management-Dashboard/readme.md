# Student Management Dashboard

Includes a web-based application that allows users to **register, manage, and display student records dynamically** using HTML, CSS, and JavaScript.

---

## 📂 Program Structure

| S. No. | Description | File |
| --- | --- | --- |
| 1. | Main dashboard interface | [`index.html`]() |
| 2. | Styling for layout and UI | [`style.css`]() |
| 3. | JavaScript logic and interactivity | [`script.js`]() |

---

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ```

2. Fill out the form and submit
3. View confirmation page

---

## 🧩 Description

The application includes:

- 📝 **Student Registration Form**
  - Name, roll number, email, department
  - Year selection (radio buttons)
  - Skills selection (checkboxes)
  - Photo upload and bio

- 📊 **Dynamic Table**
  - Displays registered student data
  - Supports edit and delete operations

- ⚙️ **JavaScript Features**
  - Form validation (name, roll, email)
  - Dynamic row creation
  - Edit and delete functionality
  - Live date & time display
  - Theme toggle (light/dark mode)

- 🎨 **UI Styling**
  - Responsive layout and styled components
