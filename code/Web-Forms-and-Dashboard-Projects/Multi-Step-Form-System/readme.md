# Multi-Step Registration & Feedback System

Includes an HTML project that implements a **multi-step registration form** with navigation across multiple pages and a final feedback section.

This project simulates a real-world form workflow where user data is collected step-by-step.

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ```

2. Fill out the form and submit
3. View confirmation page

---

## 🎯 Purpose

This collection helps in:

- Understanding multi-page form workflows
- Implementing navigation between pages
- Structuring large forms into manageable sections
- Applying CSS for consistent UI design

---

## 📌 Notes

- Uses multiple HTML pages instead of a single long form
- Navigation handled using buttons and links
- No backend (form submission is simulated)
- Suitable for intermediate web development practice
