# Application Development: Tech Event Portal (Frameset Layout)

This folder contains an HTML project that presents a **Tech Innovation Summit 2025 portal** using a frameset-based layout. The project organizes event information into multiple sections displayed simultaneously.

## 🎯 Purpose

This collection helps in:

- Understanding frameset-based web layout
- Structuring complex content into multiple sections
- Designing event or portal-style webpages
- Learning early web UI design techniques

---

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ```

2. All sections will load automatically within frames

---

## 📌 Notes

- Uses HTML `<frameset>` (deprecated in modern HTML)
- Demonstrates legacy multi-frame layout design
- Can be upgraded using modern CSS layouts (Grid/Flexbox)
