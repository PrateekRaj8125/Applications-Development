# HTML Frameset Layout

Includes HTML programs demonstrating layout design using framesets. This project shows how multiple HTML pages can be combined into a single structured interface.

---

## 📂 Program List

| S. No. | Description | Files |
| --- | --- | --- |
| 1. | Design a frameset layout using multiple HTML pages. | [`index.html`]() |
| 2. | Display institution name and logo. | [`frame1.html`]() |
| 3. | Create a navigation list using HTML. | [`frame2.html`]() |
| 4. | Display upcoming events with dates. | [`frame3.html`]() |
| 5. | Showcase highlights using styled lists. | [`frame4.html`]() |

---

## 🎯 Purpose

This collection helps in:

- Understanding frameset-based layouts
- Learning multi-page integration
- Structuring content across different sections
- Building early web layout concepts

---

## 🚀 How to Run

1. Open the main file:

    ```bash
        index.html
    ```

2. The frames will automatically load all linked pages

---

## 📌 Notes

Uses HTML `<frameset>` (deprecated in modern HTML)
Demonstrates legacy layout techniques
Useful for understanding early web design concepts
