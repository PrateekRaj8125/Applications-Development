# Frameset Layout for IoT Project

This folder contains an HTML project that demonstrates a **frameset-based layout** to present an IoT project in a structured and multi-section format.

The layout integrates multiple HTML pages into a single interface using frames.

---

## 📂 Program Structure

| S. No. | Description | Files |
| --- | --- | --- |
| 1. | Frameset layout combining all sections | [`index.html`]() |
| 2. | Project title and description | [`frame-top.html`]() |
| 3. | External project links (GitHub, LinkedIn, Demo) | [`frame2.html`]() |
| 4. | Displays project circuit image | [`frame1.html`]() |
| 5. | Shows actual project implementation | [`frame3.html`]() |

---

## 🎯 Purpose

This collection helps in:

- Understanding frameset-based layout design
- Structuring content across multiple HTML pages
- Presenting project information in a modular format
- Learning early web layout techniques

---

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ```

2. The frames will automatically load all sections

---

## 📌 Notes

- Uses HTML `<frameset>` (deprecated in modern HTML)
- Demonstrates legacy UI layout techniques
- Can be modernized using CSS Grid or Flexbox
