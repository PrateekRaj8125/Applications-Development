# Frameset Portfolio with CSS Styling

This folder contains an HTML project that demonstrates a **student portfolio website using framesets combined with modern CSS styling**.

The project integrates multiple pages (header, projects, internships) into a structured layout while applying consistent styling using an external CSS file.

---

## 📂 Program Structure

| S. No. | Description | Files |
| --- | --- | --- |

| 2. | Profile image, name, and social links | [`header.html`](https://github.com/PrateekRaj8125/Applications-Development/blob/main/code/HTML-Frameset-Projects/HTML-Frameset-Portfolio-with-CSS/header.html) |

| 4. | Internship experience details | [`internships.html`](https://github.com/PrateekRaj8125/Applications-Development/blob/main/code/HTML-Frameset-Projects/HTML-Frameset-Portfolio-with-CSS/internships.html) |

## 🎯 Purpose

This collection helps in:

- Understanding frameset-based layout design
- Combining HTML with external CSS styling
- Building structured portfolio websites
- Learning grid layouts and UI styling concepts

---

## 🚀 How to Run

1. Open the main file:

    ```text
        open index.html
    ````

2. The frames will load all sections automatically

---

## 📌 Notes

Uses HTML `<frameset>` (deprecated in modern HTML)
Demonstrates integration of CSS with multi-page layout
Can be upgraded using modern layout techniques (Flexbox/Grid without frames)
