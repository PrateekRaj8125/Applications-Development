# HTML Frameset Projects

Includes multiple HTML projects demonstrating **frameset-based layouts** for different real-world applications such as portfolios, event portals, and project presentations.

These projects focus on dividing a webpage into multiple sections and loading different HTML pages within a single interface.

## 🎯 Purpose

This collection helps in:

- Understanding frameset-based layouts
- Learning multi-page integration
- Structuring content into independent sections
- Exploring early web design techniques

---

## 📌 Notes

- Uses HTML `<frameset>` (deprecated in modern HTML)
- Useful for understanding legacy layout concepts
- Can be upgraded using modern CSS (Flexbox / Grid)
