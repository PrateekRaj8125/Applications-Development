# 🌐 Portfolio

This folder contains an HTML-based **personal portfolio webpage** that showcases basic information, projects, certifications, and technical skills.

The webpage is designed as a **simple digital resume/portfolio** using only HTML.

---

## 📄 File

- [`index.html`](https://github.com/PrateekRaj8125/Applications-Development/blob/main/code/Basic-HTML-Mini-Projects/Portfolio/index.html)

---

## 🧩 Description

The portfolio includes:

- 👤 Personal Information
  - Name, university, branch, batch details
- 🎓 Education
  - School and college background
- 🔗 Social Links
  - GitHub profile
  - LinkedIn profile
- 💻 Projects
  - IoT-based curtain automation system
  - LeetCode solutions repository
- 📜 Certifications
  - LinkedIn, Credly, Google certifications
- 🛠 Technical Skills
  - Languages: C, C++, Java, Python, JavaScript
  - Tools: MySQL, Git

Additional features:

- Profile image display
- Use of badges for links and skills
- Structured content using headings and paragraphs

---

## 🎯 Purpose

- Demonstrates creation of a personal portfolio using HTML
- Showcases academic and project details
- Provides a foundation for building professional web portfolios
- Uses external links and badge-style elements
