# 🇮🇳 Digital Museum

This folder contains an HTML-based solution that presents a **Digital Museum Exhibit of India**, showcasing historical events, inventions, famous personalities, and artifacts.

The webpage is designed to provide an informative and visually structured experience for users exploring Indian heritage.

---

## 📄 File

- [`index.html`](https://github.com/PrateekRaj8125/Applications-Development/blob/main/code/Basic-HTML-Mini-Projects/Digital-Museum/index.html)

---

## 🧩 Description

The application includes multiple sections:

- 🛕 Ancient History
  - Maurya Empire
  - Gupta Dynasty
  - British Raj & Independence
- 💡 Indian Inventions
  - Zero, Chess, Ayurveda, Cotton
- 🌟 Famous Personalities
  - Chandragupta Maurya
  - Ashoka the Great
  - Mahatma Gandhi
- 🏺 Artifacts
  - Ashoka Pillar
  - Indus Valley Seals

Additional features:

- Embedded images for visual understanding
- Styled layout using internal CSS
- Section-based organization for easy navigation

---

## 🎯 Purpose

- Demonstrates structured webpage design using HTML
- Applies basic CSS for styling and layout
- Presents educational content in an interactive format
- Combines text and media for better user engagement
