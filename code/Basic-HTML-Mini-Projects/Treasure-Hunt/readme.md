
# 🏴‍☠️ Treasure Hunt

This folder contains an HTML-based **interactive treasure hunt game**, where users navigate through multiple clues by making choices.

The game is structured as a sequence of linked webpages, creating a **decision-based adventure experience**.

---

## 📄 Files

- [`index.html`]() – Game entry point
- [`clue1.html`]() → [`clue2.html`]() → [`clue3.html`]() → [`clue4.html`]() → [`clue5.html`]() – Progressive clues
- [`trap.html`]() – Wrong choice page

---

## 🧩 Description

The application works as follows:

- 🎮 **Game Start**
  - User begins from [`index.html`]()
- 🧭 **Clue Navigation**
  - Each page presents a scenario with multiple choices
  - Correct choices lead to the next clue
  - Wrong choices redirect to a trap page
- 🏆 **Final Goal**
  - Reach the treasure by selecting the correct sequence of choices

Key features:

- Multi-page navigation using anchor links
- Decision-based progression
- Simple storytelling gameplay

---

## 🎯 Purpose

- Demonstrates linking multiple HTML pages
- Implements interactive navigation without JavaScript
- Introduces basic game logic using hyperlinks
- Enhances understanding of user flow and page transitions

---

## 🚀 How to Run

1. Open the starting file in a browser:

    ```text
        double click index.html
    ```

    OR

    ```text
        open index.html
    ```

2. Follow the clues and make choices to reach the treasure
